# School of Rogues

**School of Rogues** — a 2D Nigerian boarding school chaos game built in Godot 4.

Play as a student in a living Nigerian boarding school. Navigate strict rules, prefect patrols, teacher oversight, reputation systems, bribes, snitches, and high-stakes missions in a time-driven world full of chaos and opportunity.

## Engine
Godot 4.x (GDScript)

## MVP Goals
- Core player movement and stealth mechanics
- Time-based scheduling and NPC routines
- Reputation, heat, bribe, and punishment systems
- JSON-driven dialogue and branching narratives
- Mission framework with triggers and state
- Persistent save/load and multi-scene management
- Web export ready

## Current Status
**v0.1.0-foundation**

Clean, scalable project skeleton initialized. All directories, autoload singletons, core script placeholders, data files, and documentation are in place. No gameplay logic implemented yet. Ready for milestone-driven development.
