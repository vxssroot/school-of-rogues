# Contributing to School of Rogues

## How to Contribute
1. Fork the repo
2. Create feature branch from main
3. Commit with conventional commits
4. Open PR referencing milestone

## Code Style
- Follow official GDScript style guide
- Use class_name on all scripts
- Data-driven (JSON preferred for content)
- Lean, focused autoloads

## PR Process
- Small focused changes
- Test in Godot editor
- Update docs if needed
